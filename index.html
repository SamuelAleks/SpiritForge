About Spirit Forge

Spirit Forge is a 2D battler game developed by Alpha Games. The game traps spirits in armor and weapons and uses this gear to fight monsters. The game utilizes a peer-to-peer connection scheme to host mods, texture packs, and communication.

Features

  • Intense battles with unique spirits and gear
  • Peer-to-peer connection for hosting mods and texture packs
  • Demo available on the website
  • Ability to transfer demo data to full game
